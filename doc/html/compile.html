Compiling

under construction
  • GNU/Linux: There is a script installed at /usr/local/bin that will give you automatically the compiler and linker options that are needed to compile and link with the library. Compiler flags: 3dengfx-config --cflags Linker options: 3dengfx-config --libs
  • MS Windows:
    1. Make sure you have MS Visual Studio .NET 2003 or newer, the old VS.NET compiler won't be able to compile the code, and don't even think about VS6 no fucking chance, the compiler just sucks too much.
    2. Use the included project to build the static library
    3. link with the static library 3dengfx.lib
    4. make sure you link with the Multithreaded DLL version of the C runtime
    5. at the predefined macros define HAVE_CONFIG_H
    6. add the main directory of this library at the include search path
    7. kill a chicken with a sacrificial blade over the roaring flames of an active volcano
    8. press build...
