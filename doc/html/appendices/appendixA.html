Appendix A : Contents of this package

Here is a list of the folders contained in this package and a brief description:
  • /3dengfx : Main part of the 3d graphics engine.
  • /common : Contains miscelaneous parts of the engine that do not justify their own folder , or parts whose author has not decided where to put them yet. Currently contains image loading and curve modules , which will be transferred in the gfx section and a config file parser.
  • /doc : Documentation for the library
  • /dsys : DemoSystem. Scripting engine and ready-to-use effects.
  • /fxwt : 3dengfx Window Toolkit. SDL wrapper for callbacks. A full-accelerated window toolkit is planned.
  • /gfx : Engine-independent graphics primitives - vertex structures , triangle mesh classes etc.
  • /n3dmath2 : Nuclear's 3d math library. Vectors , matrices , quaternions , rays etc.
  • /nlibase : ASE geometry file format library
